# 2026年6月2日 AI 热点新闻

## 01 模型发布/更新

### 1. MiniMax M3 发布：100万 Token 长上下文杀入开源第一梯队

**原文总结：**MiniMax 正式发布新一代大模型 M3，支持 100 万 Token 超长上下文窗口，并采用稀疏注意力架构提升长文本推理效率。在代码能力方面，官方公布 SWE-Bench Pro 得分达到 59.0%，同时具备多模态与 Agent 调用能力。官方表示其长上下文推理速度相比前代大幅提升，并已开放 API 使用。

**个人解读：**过去一年大家一直在拼参数规模，现在开始拼“上下文长度”。100 万 Token 已经足够装下大型代码仓库、整本技术文档甚至企业知识库。如果性能数据能够被后续验证，MiniMax M3 很可能成为国内开源阵营最具竞争力的 Agent 模型之一。

**来源：** MiniMax 官方

**链接：** https://www.minimax.io/blog/minimax-m3

### 2. xAI 发布 Composer 2.5：瞄准 AI 编程赛道

**原文总结：**xAI 推出新一代编程模型 Composer 2.5，主要面向 SuperGrok 用户开放，重点强化代码生成、工程理解与开发工作流能力。该模型被定位为面向 AI Coding Agent 的专用模型，希望进一步提升复杂软件开发场景表现。

**个人解读：**从 Cursor、Claude Code 到 OpenAI Codex，如今所有头部 AI 公司都在抢夺程序员入口。xAI 此时推出 Composer 2.5，意味着 AI Coding 已经成为最核心的商业化场景之一。未来大模型竞争很可能从聊天转向开发者工作台。

**来源：** xAI

**链接：** https://x.ai/news/composer-2-5

### 3. Qwen3.7-Plus 发布：全面强化多模态 Agent 能力

**原文总结：**阿里 Qwen 团队推出 Qwen3.7-Plus，定位为多模态智能体模型。模型能够同时处理视觉、文本、GUI 界面操作以及工具调用任务，支持屏幕理解、图像推理、搜索增强和 Agent 工作流执行等能力。

**个人解读：**这反映出一个重要趋势：Agent 已经开始从“会聊天”进化为“会看屏幕、会点按钮、会操作软件”。未来的 AI 助手不只是回答问题，而是真正替用户完成任务。

**来源：** Qwen 官方

**链接：** https://qwen.ai/blog?id=qwen3.7-plus

### 4. NVIDIA 发布 Cosmos 3：物理 AI 时代正式开启

**原文总结：**NVIDIA 正式推出 Cosmos 3 开放世界基础模型。该模型首次将视觉推理、世界模拟和动作生成整合到统一架构中，能够同时理解文本、图像、视频、环境声音以及动作数据。官方表示 Cosmos 3 可用于机器人、自动驾驶和智能空间等物理 AI 场景，并开放模型权重、训练工具和数据集。

**个人解读：**过去的大模型主要服务数字世界，而 Cosmos 3 目标是让 AI 进入物理世界。机器人、自动驾驶、人形机器人等赛道都需要这类世界模型。黄仁勋最近频繁提到 Physical AI，说明 NVIDIA 已经把下一个万亿美元市场押注在机器人产业。

**来源：** NVIDIA

**链接：** https://developer.nvidia.com/blog/develop-physical-ai-reasoning-world-and-action-models-with-nvidia-cosmos-3

### 5. 商汤发布 SenseNova-U1：统一多模态理解与生成

**原文总结：**商汤推出 SenseNova-U1，采用全新的 NEO-Unify 统一架构，尝试打通视觉理解与内容生成两大能力。模型不仅具备图文理解、推理和 Agent 决策能力，还能够生成复杂图表、信息图和高质量视觉内容。官方认为这将推动多模态模型从“理解”和“生成”两个独立系统走向统一智能。

**个人解读：**图表生成只是表象，真正值得关注的是统一架构路线。未来的大模型不再区分“看图模型”和“画图模型”，而是一个模型同时具备理解和创造能力，这可能成为下一代多模态模型的发展方向。

**来源：** 商汤科技

**链接：** https://x.com/SenseTime_AI/status/2061465029959209106
